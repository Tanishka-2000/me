    Me

    I am a web developer with 5+ years of coding experience and 0 years of commercial experience. I am proficient in both frontend and backend developement. Coding is my passion and I would love to make it my profession. I believe in making simple, beautiful and interactive websites whose purpose is easily understood by user instead of a highly complex website on which users feel lost.

    My Journey

    My first programming language was C++, as a subject in school for 2 years, that's when I realised I have a talent for coding. Then I went to college for graduation in BCA where I learned C and JAVA(core) along with other computer and software developement related concepts. In the last semester I enrolled in a web developement bootcamp (and I was surprised to find out that Java and Javascript are not related in any way) and that's when my heart said "Stop!, this is it". Then my quest to become a web developer started and in a stretch of about 1.5 years I buit some really great projects (Please, do check out my portfolio. Link on the projects section).

    My Thoughts

    To me, Web developement is more art than science, where sky is the limit. Each website out there has potential to be better. As a developer, doesn't matter how good you are, you can always be better. The growth of a developer depends on his/her hunger, hunger to create an exceptional user experience, hunger to be recognised by their products, and hunger to change the world in some way through your solutions. And the most important skill for a developer is his/her ability to understand user's needs and limits and create solutions with that in mind.

    Frontend developer vs Backend developer

    The thing I love about Frontend developement is its beauty. You are as good as your imagination. I enjoy playing around with CSS a lot, there is so much you can do, based on your understanding. While the thing I love about backend is problem solving (always been a fan of puzzles) and databases (your data is never lost). Honestly, I don't have a favourite one. I just enjoy when people get amazed at what I can create.

    Projects

    These are some of the projects that I built during my learning journey. To see more projects, you can check out my portfolio.

    Inventory App

    This is a fullstack application which lets you manage your inventory for a grocery store. Home page of this app provide you with a list of categories to view. You can inspect and update any category. Basically this app let you perform all four CRUD operations on each food category and on each food item under any category. It uses node along with express to serve pages and ejs as view engine.

    Odinbook

    This is an attempt to create a facebook clone with only core functionalities. The frontend of this project is created using React and also supports client side routing, while for backend I have used node along with express to create REST APIs. This project is created using JAMstack architecture.

    Airbnb Clone

    This is a frontend project. I have tried to replicate the UI of airbnb website. It is a single page application (SPA) along with client side routing created using React. I have used a third party APIs to fetch hotels data and images. This site allows you to search hotels of any place using search bar or selecting any category from scrollbar.
